Pipeline

每条观测量按以下流水线计算（与 ObservableEngine._run_spec 一致）：

$$\text{obs}_t = \mathcal{T}^{(k)} \circ \cdots \circ \mathcal{T}^{(1)} \circ \mathcal{R}\!\left(x_t\right)$$
  • Source：从模型抓取 typed tensor $x_t$（weight / grad / activation 等）
  • Transform：逐次作用于 tensor（可选，默认 identity）
  • Reduction：tensor → 标量 $s_t$
  • Temporal：对标量序列做时序算子（可选，如 EMA / delta）

记号：$x$ 为当前 batch 上的张量；$\bar{x}$ 表示均值；$\|x\|_p$ 为 $p$-范数；$\mathrm{RMS}(x)=\sqrt{\mathbb{E}[x^2]}$。

Tensor Sources

训练轨迹上的观测对象（SourceRegistry.fetch）。

weight

参数 $\theta_t$

grad

梯度 $g_t = \nabla_\theta \mathcal{L}$

update

参数更新量 $u_t = \theta_t - \theta_{t-1}$（相邻两次 update 观测步之间）

opt_m

Adam 一阶矩 $\hat{m}_t$（exp_avg）

opt_v

Adam 二阶矩 $\hat{v}_t$（exp_avg_sq）

activation

Linear 输出激活 $a_t$，shape $(B,T,F)$ 或 $(B,F)$

preactivation

Linear 输入（激活前）$\tilde{a}_t$

logits

LM head 输出 $z_t$，shape $(B,T,C)$

attention

Causal softmax 注意力 $A_t$，shape $(B,H,T,T)$

gelu_activation

MLP GELU 后隐层，shape $(B,T,F)$

Transforms

$\mathcal{T}: \text{TypedTensor} \to \text{TypedTensor}$，按 spec 顺序依次应用。

identity

$$\mathcal{T}(x) = x$$

center

$$\mathcal{T}(x) = x - \bar{x}_{\text{sample}} \quad \text{（沿 batch 维；无 batch 则整体去均值）}$$

abs

$$\mathcal{T}(x) = |x|$$

normalize

$$\mathcal{T}(x) = \dfrac{x}{\|x\|_2 + \varepsilon} \quad \text{（沿 feature 维；无 feature 则整体归一化）}$$

square

$$\mathcal{T}(x) = x^2$$

Reductions · 基础

$\mathcal{R}: \text{TypedTensor} \to \mathbb{R}$，对 tensor 所有元素或指定轴聚合。

mean

$$\mathcal{R}(x) = \mathbb{E}[x]$$

std

$$\mathcal{R}(x) = \mathrm{Std}(x)$$

min

$$\mathcal{R}(x) = \min x$$

max

$$\mathcal{R}(x) = \max x$$

l1_norm

$$\mathcal{R}(x) = \|x\|_1 = \sum_i |x_i|$$

l2_norm

$$\mathcal{R}(x) = \|x\|_2 = \sqrt{\sum_i x_i^2}$$

rms

$$\mathcal{R}(x) = \mathrm{RMS}(x) = \sqrt{\mathbb{E}[x^2]}$$

max_abs

$$\mathcal{R}(x) = \max_i |x_i|$$

sparsity

$$\mathcal{R}(x) = \dfrac{1}{N}\sum_i \mathbf{1}[|x_i| < 10^{-6}]$$

positive_fraction

$$\mathcal{R}(x) = \dfrac{1}{N}\sum_i \mathbf{1}[x_i > 0]$$

entropy

$$p_i = \dfrac{|x_i|}{\sum_j |x_j|},\quad \mathcal{R}(x) = -\sum_i p_i \log p_i$$

Reductions · 矩阵 / 谱

主要用于 2D 权重矩阵；非方阵或维度不符时返回 NaN。

spectral_norm / top_singular_value

$$\mathcal{R}(W) = \sigma_{\max}(W)$$

trace

$$\mathcal{R}(W) = \mathrm{tr}(W) \quad \text{（要求方阵）}$$

row_std_mean

$$\mathcal{R}(W) = \dfrac{1}{r}\sum_{i=1}^{r} \mathrm{Std}(W_{i,:})$$

col_std_mean

$$\mathcal{R}(W) = \dfrac{1}{c}\sum_{j=1}^{c} \mathrm{Std}(W_{:,j})$$

effective_rank

$$X \in \mathbb{R}^{N \times F},\; \tilde{X} = X - \bar{X},\; \lambda_k = \sigma_k(\tilde{X})^2$$ $$p_k = \dfrac{\lambda_k}{\sum_j \lambda_j},\quad \mathcal{R} = \exp\!\left(-\sum_k p_k \log p_k\right)$$

Reductions · Attention

输入为 causal softmax 注意力 $A \in \mathbb{R}^{B \times H \times T \times T}$，$A_{b,h,q,k}$ 为 query $q$ 对 key $k$ 的质量。

attention_entropy_mean

$$H_{b,h,q} = -\sum_k A_{b,h,q,k}\log A_{b,h,q,k},\quad \mathcal{R} = \mathbb{E}_{b,h,q}[H_{b,h,q}]$$

attention_entropy_min

$$\mathcal{R} = \min_{b,h,q} H_{b,h,q}$$

attention_sink_first_token

$$\mathcal{R} = \mathbb{E}_{b,h,q \ge 1}[A_{b,h,q,0}] \quad \text{（非首 token query 对 key=0 的平均质量）}$$

attention_sink_domination

$$\bar{A}_k = \mathbb{E}_{b,h,q}[A_{b,h,q,k}],\quad \mathcal{R} = \max_k \bar{A}_k$$

attention_sink_ratio

$$\mathcal{R} = T \cdot \mathbb{E}_{b,h,q \ge 1}[A_{b,h,q,0}] \quad \text{（相对均匀分布 } 1/T \text{ 的倍数）}$$

Reductions · Activation / GELU

主要用于 post-GELU MLP 隐层 $x \in \mathbb{R}^{B \times T \times F}$。

activation_rate

$$\mathcal{R}(x) = \dfrac{1}{N}\sum_i \mathbf{1}[x_i > 0] \quad \text{（GELU 后 firing rate）}$$

massive_activation_peak_ratio

$$\mathcal{R}(x) = \dfrac{\max |x|}{\mathrm{RMS}(x)}$$

massive_activation_outlier_fraction

$$\mathcal{R}(x) = \dfrac{1}{N}\sum_i \mathbf{1}[|x_i| > 3\,\mathrm{RMS}(x)]$$

massive_neuron_fraction

$$p_f = \max_{b,t}|x_{b,t,f}|,\quad \mathcal{R} = \dfrac{1}{F}\sum_f \mathbf{1}[p_f > 3\,\mathrm{RMS}(x)]$$

Temporal Operators

对标量观测序列 $\{s_t\}$ 做时序变换；按 spec 顺序链式应用，各自维护内部状态。

identity

$$y_t = s_t$$

delta

$$y_t = s_t - s_{t-1} \quad \text{（首步为 NaN）}$$

ema

$$y_t = \alpha\, y_{t-1} + (1-\alpha)\, s_t \quad \text{（默认 } \alpha=0.9 \text{）}$$

slope

$$y_t = \text{OLS slope of } \{s_{t-w+1},\ldots,s_t\} \text{ vs. time index} \quad \text{（默认 } w=5 \text{）}$$

rolling_std

$$y_t = \mathrm{Std}(s_{t-w+1},\ldots,s_t) \quad \text{（默认 } w=5 \text{）}$$

curvature

$$y_t = s_t - 2s_{t-1} + s_{t-2} \quad \text{（三点二阶差分）}$$

Loss

内置观测量，无需注册 spec；每步由 observe(step, loss=...) 写入。

loss::train

$$\mathcal{R}(\ell_t) = \ell_t \quad \text{（当前 step 的 cross-entropy training loss）}$$
