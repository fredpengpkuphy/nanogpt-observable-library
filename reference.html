How to read a formula

Read each formula from top to bottom. It describes how one model tensor becomes one chart value; you do not need to memorize the symbols.

  1. Find what was captured. Xt is the complete tensor recorded at training step t—for example, a weight matrix or a batch of activations.
  2. Follow any preparation lines. A formula may center, take absolute values, normalize, or reshape Xt. If there is no such line, the tensor is used directly.
  3. Read the last line as the chart value. yt is always a single number. The chart plots the point (training step t, value yt).

Example: mean of a weight matrix

Xt = Wt ∈ ℝr×c yt = (1 / rc) Σi=1…r, j=1…c Xt,i,j

At step t, take every entry of the selected weight matrix, average them, and plot that average. Here r×c is simply the matrix shape.

When a formula contains a time-series operation

zt = reduction(Xt) yt = zt − zt⁻

First compute the tensor summary zt, then compare it with the last earlier recorded, finite value. Because observables may be sampled every 50 or 100 steps, t⁻ does not necessarily mean t−1.

Symbols used in some formulas
t
the current recorded training step
t⁻
the previous step where this observable produced a finite value
N
the number of scalar entries included in a reduction
b, h
one batch item and one attention head
q, k
one attention query position and key position
τ
a token position in the input sequence
B, T, F
batch size, sequence length, and feature width
r, c
row and column counts of a matrix
mat(X)
reshape X into a 2-D table: features are columns and all other axes become rows
